Initial project setup [@PolymerAuthors]
==============================================

```bash
cd whk.name/static
# On final site components are referenced as /assets/webcomponents/...
mkdir webcomponents
cd webcomponents
bower init
```

Install referenced components
=======================================

```bash
cd .../whk.name/static/webcomponents/
bower install Polymer/core-header-panel

```
